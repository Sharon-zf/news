# news
### 本人练习项目，持续更新中......
###### 项目名称：新闻头条（个人练习项目）
###### 技术栈：Vue 全家桶，axios，vant，socket，Eslint
###### 项目描述：该项目是一个移动端资讯类的项目，主要用于阅读 IT 类新闻，对新闻资讯进行评论等。主
###### 要模块有：新闻列表模块，登录模块，个人中心模块，文章详情模块，频道管理模块，即时通信模块。
